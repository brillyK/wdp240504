구글 [timing function]: 애니메이션 움직임 소스

[css animation]: 애니메이션 
div {
  animation-name: example; (기본값 순서 맞춰서 넣기)
  animation-duration: 5s; (기본값 순서 맞춰서 넣기)
  animation-timing-function: linear;
  animation-delay: 2s;
  animation-iteration-count: infinite;
  
}
 속기
div {
  animation: example 5s linear 2s infinite alternate;
}
